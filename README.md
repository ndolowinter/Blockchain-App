# BlockChainAppByNdolo Winter

## Description
This is a fully functional block-chain application.

## Technologies used.

* Postman App.
* Git bash.
* Atom IDE.

## Development
Run all tests:

```
$ npm install
$ npm test
$ npm run dev
```
or on Windows...
```
$ npm install
$ npm run test-windows
$ npm run dev
```
