# CYPROCOIN
This is a block-chain application that will be used to make the Cyprio-coin.

# Development
Run all tests:

```
$ npm install
$ npm test
$ npm run dev
```
or on Windows...
```
$ npm install
$ npm run test-windows
$ npm run dev
```
